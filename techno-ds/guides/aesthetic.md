# Techno aesthetic — the rules

Obsidian monochrome, a single cyan light-source. Reference: Anyma / Afterlife.
Presence comes from weight, spacing and crisp edges — never glow.

## Colour
- Ground: `--tds-bg` (#0B0C0F), `--tds-surface`, `--tds-bg-deep`.
- Text ladder: `--tds-text` (ice-white, headings/primary) → `--tds-body`
  (emotional copy) → `--tds-muted` → `--tds-label` (mono labels).
- Borders: `--tds-hairline`.
- **Cyan `--tds-accent` (#22D3EE) is rare and earned — 2-3 moments per screen,
  no bloom.** Legit uses: the hero ampersand, the sacred shloka, the headline
  event card's top-rule, a waveform tip, one interlude tick. Never on body text,
  never a per-heading highlight, never more than one cyan ampersand per screen.

## Type
- `--tds-font-display` (Space Grotesk) for everything expressive.
- `--tds-font-mono` (Space Mono), uppercase + wide tracking, for labels/eyebrows.
- `--tds-font-deva` (Noto Serif Devanagari) for the sacred shloka only.
- Extreme scale contrast: large display heads against whisper-small mono labels.

## Emphasis
- Heading emphasis is **tonal, not coloured** — wrap words in `<em>` (renders a
  lighter weight). Never colour a heading word.

## The sacred mark
- `Invocation` / `NatarajaMotif` is always centered, never tilted, warped, or
  animated. Reverent breathing room around it.

## Motion (if animating)
- Reduced-motion is a first-class path — everything must read finished static.
- No full-field luminance flashes; the cyan pulses, never strobes.
