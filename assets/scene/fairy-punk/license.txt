Note: the source model shipped as a static (unrigged, unanimated) 7-mesh
character set. It was rigged in Blender (headless, scripted) with a custom
13-bone biped armature for this project, matching the joint set the site's
procedural choreography engine already drives (see js/app/kinetic-dancer.js).
Original PBR textures/materials were NOT re-exported — this project renders
the dancer as a wireframe with a single shared material at runtime, so the
source textures are unused; stripping them keeps this asset lightweight.
